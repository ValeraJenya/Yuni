# Security

This is the initial security placeholder for Yuni.

- Do not store real secrets in the repository.
- Do not commit production credentials, API keys, tokens, private certificates, or passwords.
- Do not log tokens, passwords, session values, or unnecessary personal data.
- Keep local environment values in ignored `.env` files and document only safe examples in `.env.example`.

Full security rules will be expanded as backend, auth, chat, media, and data storage are implemented.
