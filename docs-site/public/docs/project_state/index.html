Факты baseline

Yuni - monorepo dating app. В репозитории есть:

  • frontend application;
  • NestJS backend;
  • PostgreSQL/Prisma database layer;
  • documentation;
  • Docker Compose local workflow.

Текущий стек:

  • React frontend;
  • NestJS backend;
  • PostgreSQL + Prisma;
  • Docker Compose for local development.

  • discovery;
  • likes;
